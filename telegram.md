# 🤖 Telegram Chatbot (24/7 Free Hosting)

A simple Telegram chatbot built using Node.js, hosted for **free** using Replit and kept alive with UptimeRobot.

---

## 📱 Scan to Use Bot

![Telegram Bot QR](./qr.png)

👉 Or open directly:
https://t.me/charan_keshava_bot

---

## 🚀 Features

* ✅ Responds to `/start` command
* ✅ Basic auto-replies (hi, hello, etc.)
* ✅ Runs **24/7 (near uptime)**
* ✅ Fully free (no credit card required)

---

## 🛠️ Tech Stack

* Node.js
* node-telegram-bot-api
* Express.js
* Replit (hosting)
* UptimeRobot (uptime monitoring)

---

## 📦 Installation

### 1. Clone project

```bash
git clone https://github.com/your-username/telegram-bot.git
cd telegram-bot
```

### 2. Install dependencies

```bash
npm install node-telegram-bot-api express
```

---

## 🔐 Environment Variables

Create `.env` file:

```
TOKEN=your_telegram_bot_token
```

---

## ▶️ Run the bot

```bash
node bot.js
```

---

## 🌐 24/7 Setup

### Replit

* Create Node.js repl
* Upload code
* Add TOKEN in Secrets
* Click Run

### UptimeRobot

* Add HTTP monitor
* Paste Replit URL
* Interval: 5 minutes
